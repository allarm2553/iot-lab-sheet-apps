ใบงานพื้นฐาน: การติดตั้ง Arduino IDE และคุณลักษณะของ ESP32 / ESP8266

หลักสูตรการพัฒนาระบบ Hybrid Local/Cloud IoT Node ด้วย ESP32 / ESP8266

ระบบส่งใบงานปฏิบัติการออนไลน์
คู่มือปฏิบัติการทดลอง

วัตถุประสงค์

  1. เพื่อเรียนรู้และติดตั้งโปรแกรม Arduino IDE และไดรเวอร์ USB-to-UART สำหรับบอร์ด IPST-WiFi (ESP32) และ AX-WiFi (ESP8266)
  2. เพื่อเข้าใจคุณลักษณะด้านฮาร์ดแวร์ (Specifications) ของบอร์ด IPST-WiFi และ AX-WiFi
  3. เพื่อเรียนรู้การกำหนดค่าขาและตำแหน่งพินใช้งาน (Pinout & GPIO) ของทั้งสองบอร์ด

1. การดาวน์โหลดและติดตั้ง Arduino IDE

การเขียนโปรแกรมลงบอร์ด IPST-WiFi (ESP32) หรือ AX-WiFi (ESP8266) จำเป็นต้องมีสภาพแวดล้อมในการเขียนโปรแกรม (IDE) และไดรเวอร์ติดต่อสื่อสารกับคอมพิวเตอร์ ดังนี้:

  1. ดาวน์โหลดและติดตั้ง Arduino IDE 2.x จากหน้าเว็บทางการ arduino.cc
  2. ติดตั้งไดรเวอร์เชื่อมต่อ USB-to-UART ขึ้นอยู่กับชิปควบคุมบนบอร์ดของคุณ:
    • ชิป CP2102 / CP2104 / CH340: บอร์ด IPST-WiFi และ AX-WiFi มีชิปสื่อสารติดตั้งมาแล้ว ให้ติดตั้งไดรเวอร์ที่ตรงกันเพื่อให้ระบบสืบค้น COM Port พบ
  3. เปิดโปรแกรม Arduino IDE ไปที่เมนู File → Preferences
  4. ในช่อง Additional boards manager URLs ให้คัดลอก URL ของบอร์ดที่ใช้ไปวาง (หากมีหลาย URL ให้คั่นด้วยเครื่องหมายจุลภาค `,`):
    ESP32 Board Manager URL (สำหรับ IPST-WiFi)
    ESP8266 Board Manager URL (สำหรับ AX-WiFi)
  5. ไปที่เมนูเครื่องมือด้านซ้าย เลือก Boards Manager ค้นหาคำว่า esp32 (โดย Espressif Systems) หรือ esp8266 (โดย ESP8266 Community) แล้วกดปุ่ม Install

2. คุณลักษณะทางเทคนิคของ IPST-WiFi และ AX-WiFi (Specifications)

บอร์ดไมโครคอนโทรลเลอร์ทั้งสองตัวมีสเปคทางเทคนิคและขีดความสามารถที่แตกต่างกันเพื่อตอบสนองการเลือกใช้งานตามความเหมาะสม:

คุณสมบัติ (Specification) IPST-WiFi (ESP32) AX-WiFi (ESP8266)
Processor CPU Tensilica Xtensa Dual-Core 32-bit LX6 (160 / 240 MHz) Tensilica L106 Single-Core 32-bit (80 / 160 MHz)
Memory ROM 448 KB, SRAM 520 KB + 4MB PSRAM, Flash 4MB SRAM 160 KB, Flash 4MB (บนโมดูล NodeMCU-12E)
Wireless Connectivity Wi-Fi 802.11 b/g/n (2.4 GHz) & Bluetooth v4.2 + BLE Wi-Fi 802.11 b/g/n (2.4 GHz) (ไม่มี Bluetooth)
Internal Peripherals 12-bit ADC (ต่อพอร์ต 32-35, VP=36), DAC, I2C, SPI, PWM, UART 10-bit ADC (1 ช่อง A0 ต่อ VR/AD), I2C, SPI, Software Serial, PWM
Onboard Devices OLED 0.96", LED (GPIO18), SW1 (GPIO0), RGB Neopixel x3 (GPIO12), Piezo (GPIO25) OLED 0.96", RGB WS2812B x2 (GPIO14), SW FLASH (GPIO0), Piezo (GPIO15)

3. การใช้งานพินและการจัดสรรขา (IPST-WiFi Pinout Diagram & AX-WiFi Note)

การเชื่อมต่อเซ็นเซอร์และอุปกรณ์เอาต์พุตจำเป็นต้องรู้หน้าที่ของพินแต่ละพิน คลิกเลือกพินบนรูปบอร์ดเพื่อดูรายละเอียด หรือสลับบอร์ดด้านล่าง:

โจทย์ท้าทายการทดลอง (Challenge)

โจทย์: ให้นักเรียนเขียนโปรแกรมตัวอย่างไฟกะพริบ (Blink Code) สั่งการให้ไฟ LED ที่ติดตั้งอยู่บนตัวบอร์ด (ขาพิน GPIO 2 หรือพินไฟสถานะของบอร์ด) สว่าง 1 วินาทีและดับ 1 วินาที สลับกันไปอย่างต่อเนื่อง และอัปโหลดลงบนบอร์ดทดลองจริงสำเร็จ
